
## 仙踪觅宝
东方仙侠修仙冒险游戏，玩家扮演筑基期修仙者深入神秘的灵草谷，通过战斗和探索获得修炼资源，最终挑战守护兽获得珍贵的千年灵草。采用低多边形3D美术风格，融合传统中国山水画的审美特色。

### 灵草谷
- 被群山环抱的神秘仙侠山谷，整体呈阶梯状地形从谷口向谷底逐级下降。谷口开阔平坦适合初学者试炼，谷中段地形起伏有潺潺溪流，谷底被浓郁灵雾笼罩显得神秘莫测。整个山谷弥漫着淡淡仙气，空气中闪烁着细小光点，到处生长着发光的珍贵灵草，营造出超脱尘世的仙侠氛围。地形设计体现从凡俗到仙境的层次感，越深入谷底越显神秘庄严。
- 128x128
- Spring
- Morning
- Cloudy

### Terrain
#### 谷口平地
- 椭圆形
- Size:长轴45米，短轴30米
- 地图下方中央区域，距离地图下边缘15米
- Reachable: true
- 草地地形，相对平坦开阔的椭圆形区域，地面覆盖翠绿草皮，散布着一些发光的小型灵草丛，为初级战斗提供足够的活动空间
#### 谷口巨石群
- 不规则分布
- Size:覆盖面积约400平方米
- 谷口平地内部分散分布，主要集中在左右两侧
- Reachable: true
- 土地地形，散布在谷口区域的天然巨石群，高度2-4米不等，为低级妖兽提供隐蔽和高地攻击位置，同时作为战斗中的掩体
#### 左侧山壁
- 弧形山壁
- Size:长度约90米，高度20-30米
- 地图左侧边缘，从下方延伸到上方
- Reachable: false
- 山地地形，陡峭的岩石山壁，生长着翠绿藤蔓和发光灵草，形成天然屏障，部分区域有小瀑布流下
#### 右侧山壁
- 弧形山壁
- Size:长度约90米，高度20-30米
- 地图右侧边缘，从下方延伸到上方
- Reachable: false
- 山地地形，与左侧山壁对称，同样陡峭险峻，两侧山壁共同形成山谷的天然边界
#### 谷中段丘陵地带
- 不规则多边形
- Size:面积约1800平方米
- 地图中央区域，连接谷口和谷底
- Reachable: true
- 土地和草地混合地形，地势起伏较大，有天然的小山丘和凹地，形成阶梯状过渡地带，为中级战斗提供战术地形
#### 溪流河床
- 蜿蜒带状
- Size:长度约65米，宽度4-6米
- 穿越谷中段丘陵地带，从右上向左下蜿蜒流淌
- Reachable: false
- 水面地形，清澈的山泉溪流，两岸生长着中等品质的发光灵草，水声潺潺增添仙境氛围
#### 溪流两岸平台
- 长条形平台
- Size:总面积约1000平方米
- 紧邻溪流两侧，宽度10-15米
- Reachable: true
- 草地地形，溪流冲刷形成的天然平台，地面肥沃，是中级灵草的主要生长区域，也是中级妖兽的栖息地
#### 谷底灵雾区
- 圆形
- Size:直径约40米
- 地图上方中央区域，距离地图上边缘18米
- Reachable: true
- 土壤地形，被浓郁灵雾笼罩的神秘区域，地面呈现淡金色，灵气最为浓郁，越靠近中心灵雾越浓
#### 中央石台
- 八角形
- Size:对角线长度15米，高度2.5米
- 谷底灵雾区的正中央
- Reachable: true
- 石地地形，天然形成的八角形石台，通体呈现玉石般的光泽，是千年灵草的生长地和最终Boss战的舞台，提供充足的战斗空间

### Roads
#### 谷口主道
- 起始:地图下边缘入口
- 终点:谷口平地中央
- Length:约25米
- Width:8米
- 直线形
- 材质:青石板
- 修仙者进入灵草谷的主要通道，由天然青石板铺设，两侧生长着指引性的发光灵草，路面平整适合初入者
#### 山谷步道
- 起始:谷口平地北侧
- 终点:谷中段丘陵地带南缘
- Length:约30米
- Width:8米
- 缓坡蜿蜒
- 材质:土石混合
- 连接谷口和中段的上坡小径，随地势起伏蜿蜒，路面为天然土石，体现从凡俗向仙境的过渡
#### 溪谷石桥
- 起始:溪流左岸
- 终点:溪流右岸
- Length:约10米
- Width:6米
- 拱桥形
- 材质:灵石
- 横跨溪流的天然灵石拱桥，桥身泛着微弱蓝光，是通往谷底的关键通道，石熊经常在此把守
#### 登仙小径
- 起始:谷中段丘陵地带北端
- 终点:谷底灵雾区边缘
- Length:约20米
- Width:8米
- 螺旋上升
- 材质:玉石
- 通往谷底的最后一段路程，呈缓螺旋状上升，路面由天然玉石构成，散发着淡淡仙光

### Rivers
#### 灵泉溪
- 从山顶雪融汇聚而成的清澈溪流，流水蕴含灵气，滋养着两岸的珍贵灵草，是山谷生机的源泉
- 起始:右上方山壁脚下的天然泉眼
- 终点:左下方汇入地下暗河消失
- Length:约65米
- Width:4-6米
- 从右上向左下蜿蜒流淌，在谷中段形成三个优美弯道

### Buildings
#### 古老石亭
- 谷口区域的古老石制凉亭，已有数百年历史，是修仙者休憩和观察谷内情况的地点，亭内有古老的修仙典籍石刻
- Size:占地20平方米，高度4.5米
- 谷口平地的右上方边缘，距离主道约12米
- 单体建筑，八角形结构，八根石柱支撑，顶部有传统飞檐设计
#### 溪边水榭
- 建在溪流边的简朴水榭，供路过的修仙者观水赏景，体现仙侠意境，榭内有蒲团供打坐修炼
- Size:占地15平方米，高度3.5米
- 灵泉溪左岸中段，距离溪边3米
- 单体建筑，面向溪流，三面临水一面向陆，采用传统木石结构
#### 灵草祭台
- 谷底中央石台上的神圣祭台，用于供奉和保护千年灵草，散发着神圣的金光，刻满古老符文
- Size:占地6平方米，高度2米
- 中央石台的正中心位置
- 单体建筑，圆形祭台，三层阶梯设计，刻有复杂的修仙阵法符文

### Extends
- 整个山谷被一层淡淡的灵气薄雾环绕，随风轻柔飘动，在晨光照射下呈现出七彩光晕效果
- 各品级灵草在微风中轻轻摆动，散发出不同颜色的柔和光芒：初级为淡绿色，中级为蓝色，千年灵草为金色
- 山壁上的古老藤蔓攀爬生长，部分区域有细小瀑布从高处流下汇入灵泉溪，增添仙境气息
- 谷底的浓郁灵雾会随着玩家的接近而逐渐散开，营造神秘揭晓的仪式感和Boss战前的紧张氛围
- 空气中飘浮着细小的金色光点粒子，代表浓郁的天地灵气，在战斗时会更加活跃闪烁
- 每个战斗区域都精心设计了足够的活动空间：谷口战斗区域约30x20米，中段战斗区域约25x15米，Boss战区域约15x15米
- 中央石台周围刻有古老的修仙符文阵法，在Boss战时会发出呼应的微光，增强战斗氛围的神秘感
- 整体光照采用温暖的金色调配合柔和的蓝绿色点缀，体现春晨云雾缭绕的仙境美感
- 战斗区域的地面略有起伏但不影响移动，为玩家提供丰富的战术走位选择而不造成操作障碍
