
## 仙踪觅宝
东方仙侠修仙冒险游戏，玩家扮演筑基期修仙者深入神秘的灵草谷，通过战斗和探索获得修炼资源，最终挑战守护兽获得珍贵的千年灵草。采用低多边形3D美术风格，融合传统中国山水画的审美特色，营造出超脱尘世的仙侠氛围。

### 灵草谷
- 被群山环抱的神秘仙侠山谷，整体呈阶梯状地形从谷口向谷底逐级递进。谷口开阔平坦适合初学者试炼，谷中段地形起伏有潺潺溪流环绕，谷底被浓郁灵雾笼罩显得神秘莫测。整个山谷弥漫着淡淡仙气，空气中闪烁着细小光点，到处生长着发光的珍贵灵草，体现从凡俗到仙境的层次感。地理环境融合了传统中国山水画意境，具有深厚的修仙文化底蕴，是修仙者历练成长的理想之地。

### Terrain
#### 谷口平地
- 椭圆形
- Size:长轴42米，短轴28米
- 地图下方中央区域，距离地图下边缘12米
- Reachable: true
- 草地地形，相对平坦开阔的椭圆形区域，地面覆盖翠绿草皮，散落着一些发光的小型灵草丛和天然巨石，为初级战斗提供足够的活动空间
#### 谷口巨石群
- 不规则分布
- Size:覆盖面积约350平方米
- 谷口平地内部分散分布，主要集中在左右两侧和后方
- Reachable: true
- 土地地形，散布在谷口区域的天然巨石群，高度2-4米不等，为低级妖兽提供隐蔽和攻击位置，同时作为战斗中的掩体
#### 左侧山壁
- 弧形陡壁
- Size:长度约85米，高度25-35米
- 地图左侧边缘，从下到上弧形延伸
- Reachable: false
- 山地地形，险峻的岩石山壁，生长着翠绿藤蔓和零星的发光灵草，形成天然屏障，部分区域有细小瀑布从高处流下
#### 右侧山壁
- 弧形陡壁
- Size:长度约85米，高度25-35米
- 地图右侧边缘，从下到上弧形延伸
- Reachable: false
- 山地地形，与左侧山壁对称分布，同样险峻陡峭，两侧山壁共同形成山谷的天然边界，阻隔外界
#### 谷中段丘陵地带
- 不规则带状区域
- Size:面积约1600平方米
- 地图中央偏下区域，连接谷口平地和谷底区域
- Reachable: true
- 土地和草地混合地形，地势起伏较大，有天然的小山丘和凹地，形成阶梯状过渡地带，为中级战斗提供丰富的战术地形
#### 灵泉溪河床
- 蜿蜒S形带状
- Size:长度约60米，宽度5-7米
- 穿越谷中段丘陵地带，从右上山壁向左下方向蜿蜒流淌
- Reachable: false
- 水面地形，清澈见底的山泉溪流，水中蕴含灵气，两岸生长着中等品质的发光灵草，水声潺潺营造仙境氛围
#### 溪流两岸平台
- 狭长平台群
- Size:总面积约900平方米
- 紧邻灵泉溪两侧，宽度8-12米
- Reachable: true
- 草地地形，溪流长期冲刷形成的天然平台，土壤肥沃灵气充沛，是中级灵草的主要生长区域和中级妖兽的栖息地
#### 谷底灵雾区
- 圆形
- Size:直径约38米
- 地图上方中央区域，距离地图上边缘15米
- Reachable: true
- 土壤地形，被浓郁灵雾笼罩的神秘区域，地面呈现淡金色光泽，灵气最为浓郁，越靠近中心灵雾越浓密
#### 中央石台
- 八角形平台
- Size:对角线长度14米，高度2.5米
- 谷底灵雾区的正中央
- Reachable: true
- 石地地形，天然形成的八角形石台，通体呈现玉石般的温润光泽，是千年灵草的生长地和最终Boss战的神圣舞台

### Roads
#### 入谷主道
- 起始:地图下边缘中央入口
- 终点:谷口平地中心位置
- Length:22米
- Width:8米
- 直线形
- 材质:天然青石
- 修仙者进入灵草谷的主要通道，由天然青石铺设而成，两侧生长着指引性的发光小灵草，为初入者指明方向
#### 升阶山径
- 起始:谷口平地上方边缘
- 终点:谷中段丘陵地带下方入口
- Length:28米
- Width:8米
- 缓坡蜿蜒
- 材质:土石混合
- 连接谷口和中段的上行小径，随地势缓缓上升并蜿蜒前进，路面为天然土石，体现从凡俗向仙境的自然过渡
#### 跨溪石桥
- 起始:灵泉溪左岸
- 终点:灵泉溪右岸
- Length:8米
- Width:6米
- 单孔拱桥
- 材质:青白灵石
- 横跨灵泉溪的天然灵石拱桥，桥身散发微弱蓝光，是通往谷底的关键节点，常有石熊在此驻守
#### 登仙云径
- 起始:谷中段丘陵地带上方出口
- 终点:谷底灵雾区边缘入口
- Length:18米
- Width:8米
- 螺旋缓升
- 材质:白玉石
- 通往谷底的最后一段路程，呈缓螺旋状上升，路面由天然白玉石构成，随着高度增加散发越来越强的仙光

### Rivers
#### 灵泉溪
- 源自山顶积雪融化和地下灵脉涌出的清澈溪流，流水蕴含浓郁灵气，滋养着两岸的珍贵灵草，是整个山谷生机和灵气的重要源泉
- 起始:右上方山壁脚下的天然灵泉
- 终点:左下方没入地下暗河
- Length:60米
- Width:5-7米
- 从右上向左下呈S形蜿蜒流淌，在谷中段形成三个优美的弯道，增添山水诗意

### Buildings
#### 观谷石亭
- 谷口的古老石制凉亭，历经数百年风雨，是修仙者入谷前观察情况和休憩的场所，亭内石壁刻有古老的修仙心得
- Size:占地16平方米，高度4米
- 谷口平地右侧边缘，距离入谷主道约10米处
- 单体建筑，八角亭结构，八根石柱支撑，传统飞檐造型，体现古典仙侠建筑风格
#### 临溪水榭
- 建在灵泉溪畔的简朴水榭，供修仙者观水悟道，体现山水仙侠意境，榭内置有蒲团方便打坐修炼
- Size:占地12平方米，高度3米
- 灵泉溪左岸中段，距离溪边约4米的平台上
- 单体建筑，面向溪流而建，三面临水一面向陆，采用传统木石混合结构
#### 千草祭坛
- 谷底中央石台上的神圣祭坛，专门用于供奉和守护千年灵草，散发着神圣金光，祭坛表面刻满古老的修仙符文阵法
- Size:占地8平方米，高度1.8米
- 中央石台的正中心位置
- 单体建筑，圆形三层阶梯祭坛，刻有复杂的古代修仙阵法符文，是谷中最神圣的所在

### Extends
- 整个山谷被一层若隐若现的灵气薄雾环绕，随着晨风轻柔飘动，在云层透射的阳光下呈现七彩光晕效果
- 各品级灵草在微风中轻轻摆动摇曳，散发出不同颜色的柔和光芒：初级灵草为淡绿色，中级灵草为蓝色，千年灵草为耀眼金色
- 山壁上古老的藤蔓攀爬生长覆盖，多处有细小瀑布从峭壁高处飞流而下汇入灵泉溪，增添清幽仙境气息
- 谷底的浓郁灵雾具有神秘的互动效果，会随着玩家的逐渐接近而缓缓散开，营造神秘面纱揭开的仪式感
- 空气中漂浮着无数细小的金色光点粒子，代表天地间浓郁的灵气精华，在激烈战斗时会更加活跃地闪烁跳动
- 各个战斗区域都经过精心规划确保空间充足：谷口战斗区域约30x20米，中段战斗区域约25x18米，Boss战区域约15x15米
- 中央石台周围和千草祭坛表面刻满古老的修仙符文阵法图案，在Boss战开始时会同步发出呼应的神秘微光
- 整体光照采用温暖的金黄色调为主配合柔和的青绿色点缀，完美体现春日晨雾缭绕的超脱仙境美感
- 所有战斗区域的地面都略有自然起伏但不影响正常移动，为玩家提供丰富的战术走位选择而不造成操作障碍
- 山谷各处都有天然形成的观景点和制高点，让玩家在探索过程中能够欣赏到完整的山水仙境全貌
