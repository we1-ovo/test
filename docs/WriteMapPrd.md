
## 仙踪觅宝
东方仙侠修仙冒险游戏，玩家扮演筑基期修仙者深入神秘的灵草谷，通过战斗和探索获得修炼资源，最终挑战守护兽获得珍贵的千年灵草。采用低多边形3D美术风格，融合传统中国山水画的审美特色。

### 灵草谷
- 被群山环抱的神秘仙侠山谷，整体呈阶梯状地形从谷口向谷底逐级下降。谷口开阔平坦适合初学者试炼，谷中段地形起伏有潺潺溪流，谷底被浓郁灵雾笼罩显得神秘莫测。整个山谷弥漫着淡淡仙气，空气中闪烁着细小光点，到处生长着发光的珍贵灵草，营造出超脱尘世的仙侠氛围。地形设计体现从凡俗到仙境的层次感，越深入谷底越显神秘庄严。
- 128x128
- Spring
- Morning
- Cloudy

### Terrain
#### 谷口平地
- 椭圆形
- Size:长轴40米，短轴25米
- 地图下方偏左区域，距离地图边缘15米
- Reachable: true
- 草地地形，相对平坦开阔的椭圆形区域，地面覆盖翠绿草皮，散布着一些发光的小型灵草丛，营造仙侠入门试炼的氛围
#### 谷口巨石群
- 不规则分布
- Size:覆盖面积约300平方米
- 谷口平地内部和周边，分散分布
- Reachable: true
- 土地地形，散布在谷口区域的天然巨石群，高度2-4米不等，为低级妖兽提供隐蔽和高地攻击位置
#### 左侧山壁
- 弧形山壁
- Size:长度约80米，高度15-25米
- 地图左侧边缘，从下方延伸到上方
- Reachable: false
- 山地地形，陡峭的岩石山壁，生长着翠绿藤蔓和发光灵草，形成天然屏障
#### 右侧山壁
- 弧形山壁
- Size:长度约80米，高度15-25米
- 地图右侧边缘，从下方延伸到上方
- Reachable: false
- 山地地形，与左侧山壁对称，同样陡峭险峻，两侧山壁共同形成山谷的天然边界
#### 谷中段丘陵地带
- 不规则多边形
- Size:面积约1500平方米
- 地图中央偏下区域，连接谷口和谷底
- Reachable: true
- 土地和草地混合地形，地势起伏较大，有天然的小山丘和凹地，形成阶梯状过渡地带
#### 溪流河床
- 蜿蜒带状
- Size:长度约60米，宽度3-5米
- 穿越谷中段丘陵地带，从右上向左下流淌
- Reachable: false
- 水面地形，清澈的山泉溪流，两岸生长着中等品质的发光灵草，水声潺潺增添仙境氛围
#### 溪流两岸平台
- 长条形平台
- Size:总面积约800平方米
- 紧邻溪流两侧，宽度8-12米
- Reachable: true
- 草地地形，溪流冲刷形成的天然平台，地面肥沃，是中级灵草的主要生长区域
#### 谷底灵雾区
- 圆形
- Size:直径约35米
- 地图上方中央区域，距离地图边缘20米
- Reachable: true
- 土壤地形，被浓郁灵雾笼罩的神秘区域，地面呈现淡金色，灵气最为浓郁
#### 中央石台
- 八角形
- Size:对角线长度12米，高度2米
- 谷底灵雾区的正中央
- Reachable: true
- 石地地形，天然形成的八角形石台，通体呈现玉石般的光泽，是千年灵草的生长地和最终Boss战的舞台

### Roads
#### 谷口主道
- 起始:地图下边缘入口
- 终点:谷口平地中央
- Length:约20米
- Width:8米
- 直线形
- 材质:青石板
- 修仙者进入灵草谷的主要通道，由天然青石板铺设，两侧生长着指引性的发光灵草
#### 山谷步道
- 起始:谷口平地
- 终点:谷中段丘陵地带
- Length:约35米
- Width:8米
- 蜿蜒上坡
- 材质:土石混合
- 连接谷口和中段的上坡小径，随地势起伏蜿蜒，路面为天然土石，体现从凡俗向仙境的过渡
#### 溪谷石桥
- 起始:溪流左岸
- 终点:溪流右岸
- Length:约8米
- Width:6米
- 拱桥形
- 材质:灵石
- 横跨溪流的天然灵石拱桥，桥身泛着微弱蓝光，是通往谷底的关键通道，经常有石熊把守
#### 登仙小径
- 起始:谷中段丘陵地带
- 终点:谷底灵雾区边缘
- Length:约25米
- Width:8米
- 螺旋上升
- 材质:玉石
- 通往谷底的最后一段路程，呈螺旋状上升，路面由天然玉石构成，散发着淡淡仙光

### Rivers
#### 灵泉溪
- 从山顶雪融汇聚而成的清澈溪流，流水蕴含灵气，滋养着两岸的珍贵灵草
- 起始:右上方山壁脚下的泉眼
- 终点:左下方汇入地下暗河
- Length:约60米
- Width:3-5米
- 从右上向左下蜿蜒流淌，在谷中段形成几个小弯道

### Buildings
#### 古老石亭
- 谷口区域的古老石制凉亭，已有数百年历史，是修仙者休憩和观察谷内情况的地点
- Size:占地16平方米，高度4米
- 谷口平地的右侧边缘，距离巨石群约8米
- 单体建筑，八角形结构，石柱支撑
#### 溪边水榭
- 建在溪流边的简朴水榭，供路过的修仙者观水赏景，体现仙侠意境
- Size:占地12平方米，高度3米
- 灵泉溪左岸中段，紧邻溪流边缘
- 单体建筑，面向溪流，三面环水
#### 灵草祭台
- 谷底中央石台上的小型祭台，用于供奉和保护千年灵草，散发着神圣的金光
- Size:占地4平方米，高度1.5米
- 中央石台的正中心位置
- 单体建筑，圆形祭台，刻有古老符文

### Extends
- 整个山谷被一层淡淡的灵气薄雾环绕，随风飘动，增强仙侠氛围感
- 灵草在微风中轻轻摇摆，散发出不同颜色的柔和光芒，形成梦幻的视觉效果
- 山壁上的藤蔓攀爬生长，部分区域有瀑布从高处流下汇入灵泉溪
- 谷底的灵雾会随着玩家的接近而逐渐散开，营造神秘揭晓的仪式感
- 空气中飘浮着细小的光点粒子，代表浓郁的天地灵气
- 各个战斗区域都有足够的空间供玩家施展仙术，地面略有起伏但不影响移动
- 石台周围刻有古老的修仙符文，在Boss战时会发出微光呼应战斗氛围
- 整体光照偏向温暖的金色调，体现晨光初现的仙境美感
